Перечисления (Enum)
---------------------

Перечисление Java, также называемое типом перечисления Java,
представляет собой тип, поля которого состоят из фиксированного
набора констант.

Перечисления служат для представления группы именованных
констант.

Например, 4 масти в колоде игральных карт могут быть
4 перечислениями с именами Club (Трефы), Diamond (Бубны),
Heart (Червы/черви)
и Spade (Пики), принадлежащими к пронумерованному типу с
именем Suit.

Другие примеры включают естественные нумерованные типы
(например, дни недели, цвета, направления и т.п.).

Перечисления используются, когда мы знаем все возможные
значения во время компиляции, такие как варианты выбора в меню,
режимы округления и т.п.

В Java (начиная с версии 1.5) перечисления представлены с
использованием типа данных enum.

Основная цель enum - определить наши собственные типы данных
(перечисляемые типы данных).
